ℹ️ 使用说明

    

      
第一步：点击「连接 ESP32」，选择 ESP32-Setup

      
第二步：点击「扫描 WiFi」，选择你的网络并输入密码

      
第三步：点击「连接 WiFi」，等待 ESP32 反馈 wifi_ok

      
OTA 更新：WiFi 连接成功后，在 Arduino 串口监视器输入 1 回车触发 OTA

    


    


    
📋 状态说明

    

      
scanning → 正在扫描 WiFi

      
scan_done → 扫描完成

      
wifi_connecting → 正在连接 WiFi

      
wifi_ok → WiFi 连接成功

      
wifi_failed → WiFi 连接失败
